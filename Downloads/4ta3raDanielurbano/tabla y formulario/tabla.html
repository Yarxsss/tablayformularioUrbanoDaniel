¿Que es una tabla html y para que sirve?

Una tabla html es un medio para organizar datos en fila y columnas, este concepto fue adoptado por HTML como una forma de transmitir informacion de manera resumida como un grupo de filas donde cada una contiene un grupo de celdas

.Cuales son las etiquetas html (tags) básicas ?
